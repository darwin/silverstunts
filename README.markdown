SilverStunts - a data driven game in SilverLight
===================

![intro][intro]

Features:
---------

* Physics-driven 2D game
* Scrolling
* Levels loaded on demand
* In-game editor
* Live scripting with Python on client-side (using HTML script editor)
* Game objects skinnable with XAML templates
